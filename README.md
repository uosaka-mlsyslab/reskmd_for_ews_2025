# Generalized Stochastic Resilience for Early Warning Signals Based on Koopman Operator
This repository contains code to reproduce results in the preprint,

Y. Miyauchi, M. Ikeda, and Y. Kawahara, "Generalized Stochastic Resilience for Early Warning Signals Based on Koopman Operator", arXiv:2508.19655, 2025.

## Requirements
The implementation is designed to run in a Python 3.9.13 environment.
The necessary libraries are as follows:
- matplotlib                   3.9.4
- numpy                        1.26.4
- pandas                       2.3.1
- scikit-learn                 1.6.1
- scipy                        1.13.1
- tensorflow                   2.15.0

## Data sources
The datasets and deep learning models used in this study are available from the following github pages: https://github.com/ThomasMBury/deep-early-warnings-pnas.
